## рҹҡҖ Getting Started

### Prerequisites

- Node.js вүҘ 20.0
- npm вүҘ 10.0
- TypeScript вүҘ 5.3

### Installation

```bash
# Clone the repository
git clone https://github.com/KOSASIH/PiNexus-Banking-Nexus.git
cd PiNexus-Banking-Nexus

# Install dependencies
npm install

# Build
npm run build

# Run local testnet
npm run testnet

# Start neural mining (development mode)
npm run mine:dev
```

### Quick Start

```typescript
import {
  TransformerEngine,
  RLHFEngine,
  RAGEngine,
  MultiModalEngine,
  FederatedLearningEngine,
  AIAgent,
  HybridDualCoinService,
  SecurityFortress,
} from 'pinexus';

// Initialize transformer with MoE
const transformer = new TransformerEngine({
  modelDim: 8192,
  numLayers: 128,
  attention: { numHeads: 64, numKVHeads: 8, headDim: 128, maxSeqLen: 131072, ... },
  moe: { numExperts: 64, activeExperts: 8, routerType: 'top_k', ... },
});

// Generate text
const result = await transformer.generate({
  id: 'req-1',
  tokens: [1, 2, 3],
  maxNewTokens: 512,
  temperature: 0.7,
  topP: 0.9,
  priority: 'high',
});

// RAG query with multi-hop reasoning
```
